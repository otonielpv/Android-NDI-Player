# Librerías NDI Nativas

Este directorio debe contener las librerías NDI del SDK oficial para cada arquitectura:

```
jniLibs/
├── arm64-v8a/
│   └── libndi.so
├── armeabi-v7a/
│   └── libndi.so  
├── x86/
│   └── libndi.so
└── x86_64/
    └── libndi.so
```

Copia las librerías del SDK NDI que descargaste en estos directorios.
